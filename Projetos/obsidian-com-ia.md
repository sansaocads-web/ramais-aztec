# Obsidian com Inteligência Artificial

Projeto para integrar ferramentas de IA ao fluxo de trabalho no Obsidian, automatizando a criação de notas, resumos e conexões entre ideias.

## Objetivo

Construir um sistema de gestão do conhecimento pessoal (PKM) potencializado por IA, reduzindo o atrito na captura e conexão de informações.

## Etapas do Roteiro

- [ ] Definir quais plugins de IA usar no Obsidian
- [ ] Testar integração com [[ferramenta-chatgpt]] via API
- [ ] Testar integração com [[ferramenta-claude]] via API
- [ ] Criar templates automatizados de notas diárias
- [ ] Configurar sumarização automática de notas longas
- [ ] Conectar com o projeto [[automacao-n8n]] para fluxos automáticos
