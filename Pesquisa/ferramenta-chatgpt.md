# ChatGPT

Modelo de linguagem da OpenAI, disponível via interface web e API. Um dos LLMs mais populares para geração de texto, código e análise.

## Características Principais

- **Modelos disponíveis:** GPT-4o, GPT-4 Turbo, GPT-3.5
- **Acesso:** Web (chat.openai.com), API REST, plugins
- **Pontos fortes:** Geração de texto criativo, código, sumarização
- **Limitações:** Pode alucinar fatos; contexto limitado em versões antigas

## Casos de Uso Relevantes

- Geração de rascunhos de notas para o vault [[obsidian-com-ia]]
- Criação de nós de automação no projeto [[automacao-n8n]]
- Resposta rápida a dúvidas durante pesquisas
