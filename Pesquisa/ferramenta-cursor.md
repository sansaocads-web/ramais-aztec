# Cursor

Editor de código baseado em VSCode com IA integrada nativamente. Permite completar, refatorar e gerar código por meio de linguagem natural diretamente no editor.

## Características Principais

- **Tipo:** IDE com IA integrada (fork do VSCode)
- **Acesso:** Desktop (cursor.sh), assinatura gratuita e paga
- **Pontos fortes:** Autocompletar com contexto do projeto inteiro, chat com o código, geração por linguagem natural
- **Limitações:** Focado em código; menos útil para escrita de texto puro

## Casos de Uso Relevantes

- Desenvolvimento de scripts de automação para o projeto [[automacao-n8n]]
- Criação de plugins ou scripts para integrar IA ao [[obsidian-com-ia]]
- Edição rápida de templates e arquivos de configuração do vault
