# Perplexity AI

Motor de busca potencializado por IA que retorna respostas com citações de fontes em tempo real. Diferente dos LLMs tradicionais, o Perplexity sempre indica de onde vem a informação.

## Características Principais

- **Tipo:** Motor de busca + LLM com acesso à web em tempo real
- **Acesso:** Web (perplexity.ai), API, app mobile
- **Pontos fortes:** Respostas com fontes verificáveis, atualização em tempo real
- **Limitações:** Menos criativo que GPT/Claude para geração de conteúdo longo

## Casos de Uso Relevantes

- Pesquisa ativa para enriquecer notas do vault [[obsidian-com-ia]]
- Coleta de referências atualizadas para os projetos [[automacao-n8n]] e [[obsidian-com-ia]]
- Verificação de fatos antes de salvar informações como notas permanentes
