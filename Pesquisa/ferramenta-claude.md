# Claude — Anthropic

Modelo de linguagem da Anthropic, focado em segurança e raciocínio de longa extensão. Destaque para análise profunda de documentos e tarefas que exigem atenção a detalhes.

## Características Principais

- **Modelos disponíveis:** Claude Opus 4.7, Claude Sonnet 4.6, Claude Haiku 4.5
- **Acesso:** Web (claude.ai), API REST, Claude Code (CLI)
- **Pontos fortes:** Análise de documentos longos, raciocínio passo a passo, código
- **Limitações:** Sem acesso nativo à web em tempo real

## Casos de Uso Relevantes

- Análise e sumarização de notas longas no [[obsidian-com-ia]]
- Classificação de conteúdo nos fluxos do [[automacao-n8n]]
- Revisão de código e explicação de conceitos
